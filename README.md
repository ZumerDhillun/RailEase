# RailEase – Railway Reservation System 

##  Overview
RailEase is a command-line based Railway Reservation System developed in C++. The system is designed to simulate real-world railway operations, including ticket reservations, train management, and route optimization.

It demonstrates the practical implementation of core Data Structures and Algorithms concepts such as linked lists, queues, and graph-based shortest path algorithms.

---

##  Key Features

###  User Panel
- Reserve train tickets  
- View existing reservations  
- Cancel booked tickets  
- Edit reservation details  
- Select preferred seats  
- Find the shortest route between stations  

---

###  Admin Panel
- Secure login with credential verification  
- Add new trains to the system  
- Depart trains (queue-based handling)  
- Search trains by ID or route  
- View trains by specific routes  
- Display all train records  

---

##  Data Structures & Algorithms Used

- **Linked Lists**  
  Used to manage dynamic records of trains and reservations  

- **Queue**  
  Handles train departures in a structured FIFO manner  

- **Map (STL)**  
  Stores and retrieves route and train-related data efficiently  

- **Graph Algorithms (Shortest Path)**  
  Used to calculate the most efficient route between stations  

---

##  Technologies Used
- **Language:** C++  
- **Concepts:** Data Structures & Algorithms  
- **Libraries:** STL (map, queue), iostream, string, algorithm  
- **Platform:** Command Line Interface (CLI)  

---

##  System Workflow
1. Users interact with the system through the CLI  
2. Reservations are stored and managed using linked lists  
3. Admin manages train schedules and records  
4. Queue structure processes train departures  
5. Graph algorithm calculates shortest paths between stations
